# 操作规范 — Standards

> 工程运行时必须遵守的规则。新 Agent 接入必须先读。

## 数据库操作（如有）

| 原则 | 说明 |
|------|------|
| **读共享，写独占** | 所有 agent 可同时读，写操作需按规范执行 |
| **只读引用** | 本工程只读引用外部库，不持有副本（或写明：只读不写） |
| **写后必须备份** | 任何修改前先备份原文件 |
