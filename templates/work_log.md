# 工作记录 — Work Log

> 多 Agent / 多人协作工程。每次推进后更新，署全名，写清楚。

## 格式

```
YYYY-MM-DD HH:MM | 执行人 | 做了什么 | 结果 | 备注
```

- **执行人**：写全名（{Agent1} / {Agent2} / 人名）
- **做了什么**：具体操作，不要模糊
- **结果**：✅ 成功 / ⚠️ 部分成功 / ❌ 失败 + 关键数据
- **备注**：注意事项、依赖条件、待办项、需要谁做什么
